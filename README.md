# 📘 Projet Web – Formation

## 👤 Auteur

**Nom :** Gafsi Foued

---

## 📌 Description

Ce projet a été réalisé dans le cadre de ma formation en développement web.
L'objectif principal est de mettre en pratique les concepts fondamentaux du développement web ainsi que l'intégration de fonctionnalités modernes.

---

## ⚙️ Fonctionnalités réalisées

### 🏗️ Architecture

* Implémentation du modèle **MVC (Modèle - Vue - Contrôleur)**
* Organisation claire et structurée du code

### 🔍 Fonctionnalités principales

* Recherche dynamique
* Filtrage des données
* Pagination des résultats

### 💬 Communication & Interaction

* Système de chat en **JavaScript**
* Envoi d'emails via **SMTP**

### 🌐 Intégrations API

* API de traduction automatique
* API de filtrage des mots inappropriés (bad words)
* Utilisation des API **Gemini** pour des fonctionnalités intelligentes

---

## 🛠️ Technologies utilisées

* PHP
* JavaScript
* HTML / CSS
* SMTP (email sending)
* APIs externes

---

## 🎯 Objectifs pédagogiques

* Comprendre et appliquer le modèle MVC
* Manipuler les APIs externes
* Améliorer l'expérience utilisateur avec JavaScript
* Gérer les données avec des fonctionnalités avancées (recherche, filtre, pagination)

---

## 📎 Remarques

Ce projet représente une synthèse des compétences acquises durant la formation, avec une attention particulière portée à la structuration du code et à l’intégration de services modernes.
